# JavaScript DOM & OOP Assignments Hub

A single-page web application that combines five distinct JavaScript programming assignments covering Object-Oriented Programming (OOP), Document Object Model (DOM) manipulation, Event Listeners, JSON parsing, and Regular Expressions (Regex).

---

## 🚀 Features & Task Breakdown

### Assignment 1
* **Task 1: Library Management System**
    * Built using JavaScript Classes (`Book` and `Library`).
    * Simulates adding, borrowing, returning, and listing books.
    * *Note: Outputs results dynamically to the browser's developer console.*
* **Task 2: Dynamic Shopping Cart**
    * Renders a product catalog dynamically from data arrays.
    * Features real-time state management to add/remove items and calculate a running total price automatically.
* **Task 3: Interactive Quiz App**
    * Presents a multiple-choice quiz one question at a time.
    * Provides instant visual feedback by changing the application background to **green** (correct) or **red** (incorrect) before advancing.
    * Displays final score and automated grade remarks upon completion.

### Assignment 2
* **Task 4: JSON Data Processing**
    * Parses a raw nested JSON string containing student grade profiles across two classes.
    * Computes individual student averages across three core subjects (`CSC101`, `CSC102`, `CSC103`).
    * Renders an organized results table with dynamically calculated Pass/Fail badges (Pass mark: 50).
* **Task 5: Real-Time Registration Form Validator**
    * Implements safe client-side input checking utilizing Regular Expressions (Regex).
    * Validates custom constraints for alphanumeric usernames, robust passwords, and standard Nigerian mobile formatting (`+234`).
    * Injects instant, live textual feedback directly next to input fields as the user types.

## 📦 How to Setup and Run

1.  **Clone or Move Files:** Ensure both `index.html` and your main script file (e.g., `app.js` or `Cart.js`) are placed inside the **same directory folder**.
2.  **Open the Application:** Open `index.html` using a modern mobile code viewer or web browser.
3.  **Run Preview:** Click **Play/Preview** within your IDE to interact with the live shopping cart, quiz, data tables, and form validator inputs.
4.  **Check Console Logging:** Open your browser or IDE's built-in **Console panel** to inspect the step-by-step verification logs generated automatically by the Class-based Library System.
